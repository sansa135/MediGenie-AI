# 🌟 About The Project

MediGenie AI is an advanced AI-powered healthcare assistant that predicts diseases using symptoms and machine learning.

The system analyzes user symptoms, predicts possible diseases, and provides intelligent healthcare recommendations through a modern interactive web application.

---

# 🔥 Core Features

## 🩺 Disease Prediction Engine
- Predicts diseases using Machine Learning
- Supports 100+ symptoms
- Fast and intelligent analysis

## 🤖 AI Medical Assistant

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/sansa135/MediGenie-AI.git

cd MediGenie-AI
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Train AI Model

```bash
python train_model.py
```

---

## Run Application

```bash
python app.py
```
